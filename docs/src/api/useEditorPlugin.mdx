# useEditorPlugin

> `useEditorPlugin(editor plugin);`

`useEditorPlugin` adds a plugin to an editor instance

```tsx
import { useEditor, useEditorPlugin } from '@sagold/react-json-editor';

function WebFormComponent() {
  const editor = useEditor(options)
  useEditorPlugin(editor, plugin);

  // ...
```
