# Cookbook

> Examples on how to build complex web forms using `@sagold/react-json-editor` and `@sagold/rje-mantine-widgets`

- [**Conditional Forms**](?path=/docs/cookbook-conditionalforms--docs) Show parts of form conditionally and retain data when toggling the condition
- [**Drag&Drop Layout**](?path=/story/cookbook-dragdroplayout--drag-drop-layout) Drag and drop form elements to configure a layout
- [**Drag&Drop Lists**](?path=/story/cookbook-dragdroplists--drag-drop-lists) Drag and drop between two lists. Including drop validation
- [**Drag&Drop Template**](?path=/story/cookbook-dragdroptemplate--drag-drop-template) Drag from a list of templates to add data
- [**Portal Editor**](?path=/story/cookbook-dataportal--data-portal) Nesting of two editor instances to work on multiple data sets in one form
- [**Select For Detail**](?path=/story/cookbook-selectfordetails--select-for-details) Show form details when selecting an item
- [**Undo/Redo**](?path=/story/cookbook-undoredo--undo-redo) Undo/Redo support for user actions
- [**Login Form**](?path=/story/cookbook-login--login) Custom styled login form

For further details refer to the corresponding [stories on github](https://github.com/sagold/json-editor/tree/main/docs/src/cookbook)
