# JSON Schema behaviour

A json-editor has to be **opinionated** in some cases, when using JSON Schema to describe a web forms. This section details the behaviour for most JSON Schema keywords. Note that the opinionated behaviour is handled by the corresponding widget. Simply replace a given widget by your **own widget to solve your use cases**.
